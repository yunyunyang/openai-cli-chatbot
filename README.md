# CLI AI Chatbot

A lightweight, customizable Command Line Interface (CLI) AI chatbot built with the OpenAI Python SDK. Switch between different personas by tweaking the system configuration.

---

## Quick Start

Follow these simple steps to set up and run the chatbot on your local machine.

### 1. Install Dependencies

First, install the official OpenAI Python library using `pip`:

```bash
pip install openai
```

### 2. Set Up Environment Variables

Configure your API credentials in your terminal session:

```bash
export API_KEY="Your API Key"
export BASE_URL="https://api.openai.com/v1"
```

### 3. Run the Chatbot

Launch the script to begin your interactive terminal chat:

```bash
python chatbot.py
```
